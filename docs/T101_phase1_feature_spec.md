## 개요

Phase 1은 **수집 및 도감 (MVP)** 단계로, 서버 없이 로컬 저장소만을 활용하여 핵심 기능을 구현한다.

### 핵심 기능 요약
1. **디지털 펀치** - 이미지에서 원하는 영역을 프레임 모양으로 크롭
2. **도감 등록** - 크롭한 이미지에 메타데이터를 붙여 아카이빙
3. **폴더 분류** - 대분류/소분류 폴더 구조로 체계적 정리
4. **테마 설정** - 복숭아 우유/민트 소다/포도 캔디 3가지 테마 선택

## Feature 1: 디지털 펀치 (Image Punch Cropper)

| 항목 | 내용 |
|------|------|
| **기능 ID** | F1-PUNCH |
| **목적** | 사진에서 원하는 영역을 다양한 프레임 모양으로 크롭 |

### 세부 기능

| ID | 기능명 | 설명 | 입력 | 출력 |
|----|--------|------|------|------|
| F1.1 | 이미지 불러오기 | 카메라 촬영 또는 갤러리에서 이미지 선택 | 사용자 액션 | 원본 이미지 |
| F1.2 | 프레임 선택 | 엽서(직사각형), 우유곽, 원형, 사각형 등 프리셋 프레임 제공 | 프레임 타입 | 마스크 오버레이 |
| F1.3 | 프레임 위치/크기 조절 | 드래그로 위치 이동, 핀치로 크기 조절 | 제스처 입력 | 조정된 프레임 영역 |
| F1.4 | 크롭 실행 | 선택 영역을 투명 배경 PNG로 추출 | 확정 버튼 | 크롭된 이미지 (PNG) |
| F1.5 | 햅틱/사운드 피드백 | 크롭 시 "탁-터덕" 사운드 + 진동 | 크롭 실행 | 감각 피드백 |

### 기술 요구사항
- `image_picker` 패키지 (카메라/갤러리 접근)
- 커스텀 마스킹 로직 (`image_cropper`는 사각형/원형만 지원하여 하트·우유곽·폴라로이드 등 비정형 프레임에 부적합 → 채택 안 함)
- `CustomClipper<Path>` + `GestureDetector`/`InteractiveViewer`로 화면상 프레임 오버레이(드래그/핀치) 구현
- `image` 패키지로 실제 픽셀 단위 크롭 + 알파 마스크 적용 후 투명 PNG 인코딩 (F1.4 최종 출력)
- `HapticFeedback` API + `audioplayers` 패키지

### 프레임 프리셋 목록 (MVP)
| 프레임명 | 형태 | 비율/크기 |
|----------|------|-----------|
| 엽서 | 둥근모서리 직사각형 | 3:2 |
| 우유곽 | 사다리꼴 상단 직사각형 | 커스텀 |
| 원형 | 정원 | 1:1 |
| 정사각형 | 정사각형 | 1:1 |
| 폴라로이드 | 하단 여백 직사각형 | 커스텀 |

## Feature 2: 도감 등록 시스템 (Collection Registry)

| 항목 | 내용 |
|------|------|
| **기능 ID** | F2-REGISTRY |
| **목적** | 크롭한 이미지를 메타데이터와 함께 도감에 등록 |

### 세부 기능

| ID | 기능명 | 설명 | 입력 | 출력 |
|----|--------|------|------|------|
| F2.1 | 자동 넘버링 | [No.001] 형식으로 순차 번호 자동 부여 | 등록 액션 | 고유 도감 번호 |
| F2.2 | 카테고리 라벨링 | 사용자 정의 태그 추가 (예: #카페, #빈티지) | 텍스트 입력 | 라벨 배열 |
| F2.3 | 별점 평가 | 5점 만점 별점 등록 | 별 선택 (1~5) | 정수 rating |
| F2.4 | 한 줄 메모 | 짧은 설명 또는 감상 기록 | 텍스트 입력 | 메모 문자열 |
| F2.5 | 수집 일시 기록 | 등록 시점 자동 저장 | 시스템 시간 | timestamp |
| F2.6 | 썸네일 생성 | 리스트 표시용 축소 이미지 자동 생성 | 원본 이미지 | 썸네일 이미지 |

### 데이터 모델

```dart
class CollectionItem {
  String id;           // UUID
  int docNumber;       // 도감 번호 (No.001)
  String imagePath;    // 로컬 이미지 경로
  String thumbnailPath;
  List<String> labels; // 카테고리 태그
  int rating;          // 1~5
  String memo;         // 한 줄 메모
  DateTime createdAt;
  String? folderId;    // 소속 폴더 (null이면 미분류)
}
```

## Feature 3: 도감 조회/탐색 (Collection Viewer)

| 항목 | 내용 |
|------|------|
| **기능 ID** | F3-VIEWER |
| **목적** | 등록된 도감 아이템을 다양한 방식으로 탐색 |

### 세부 기능

| ID | 기능명 | 설명 |
|----|--------|------|
| F3.1 | 그리드 뷰 | 썸네일 기반 격자형 목록 (2~3열) |
| F3.2 | 리스트 뷰 | 상세 정보 포함 세로 목록 |
| F3.3 | 상세 보기 | 풀스크린 이미지 + 전체 메타데이터 표시 |
| F3.4 | 정렬 옵션 | 최신순 / 번호순 / 별점순 |

- 기본 뷰: 2열 그리드 (썸네일 + 도감번호)
- 토글로 3열 그리드 / 리스트 뷰 전환
- 상단 검색바 + 필터 칩

## Feature 4: 폴더 분류 시스템 (Folder Management)

| 항목 | 내용 |
|------|------|
| **기능 ID** | F4-FOLDER |
| **목적** | 대분류/소분류 폴더로 수집품 체계적 정리 |

### 세부 기능

| ID | 기능명 | 설명 |
|----|--------|------|
| F4.1 | 폴더 생성 | 이름, 아이콘/컬러 지정하여 폴더 생성 |
| F4.2 | 계층 구조 | 대분류 폴더 내 소분류 폴더 (최대 2depth) |
| F4.3 | 아이템 이동 | 드래그앤드롭 또는 선택 후 이동 |
| F4.4 | 폴더 편집 | 이름 변경, 삭제, 병합 |
| F4.5 | 폴더별 통계 | 폴더 내 아이템 개수, 최근 추가일 표시 |

### 데이터 모델

```dart
class Folder {
  String id;           // UUID
  String name;
  String? parentId;    // null이면 루트(대분류) 폴더
  String? iconEmoji;   // 폴더 아이콘 (이모지)
  String? colorHex;    // 폴더 테마 컬러
  DateTime createdAt;
  int itemCount;       // 하위 아이템 수 (computed)
}
```

## Feature 5: 로컬 데이터 저장 (Local Persistence)

| 항목 | 내용 |
|------|------|
| **기능 ID** | F5-STORAGE |
| **목적** | 오프라인 환경에서 모든 데이터를 기기 내 저장 및 안전하게 백업 |

### 세부 기능

| ID | 기능명 | 설명 |
|----|--------|------|
| F5.1 | 이미지 저장 | 앱 전용 디렉토리에 원본/썸네일 저장 |
| F5.2 | 메타데이터 DB | SQLite(sqflite)로 구조화된 데이터 관리 |
| F5.3 | 자동 로컬 백업 | 앱 사용 중 주기적으로 자동 백업 생성 |
| F5.4 | 데이터 내보내기 | 전체 데이터를 ZIP 파일로 내보내기 (공유 가능) |
| F5.5 | 데이터 가져오기 | 백업 ZIP 파일에서 데이터 복원 |

### 백업 시스템 상세

#### F5.3 자동 로컬 백업
| 항목 | 내용 |
|------|------|
| 백업 주기 | 앱 종료 시 + 매일 1회 (백그라운드) |
| 보관 개수 | 최근 3개 백업 유지 (자동 순환 삭제) |
| 백업 위치 | 앱 내부 `/backups/` 디렉토리 |
| 백업 형식 | JSON (메타데이터) + 이미지 파일 참조 |

#### F5.4 데이터 내보내기 (Export)
| 항목 | 내용 |
|------|------|
| 내보내기 형식 | `.pongdang` 확장자 ZIP 파일 |
| 포함 내용 | 모든 이미지 + 메타데이터 JSON + 폴더 구조 |
| 공유 방식 | 시스템 공유 시트 (카톡, 메일, 드라이브 등) |
| 파일명 규칙 | `pongdang_backup_YYYYMMDD_HHMMSS.pongdang` |

#### F5.5 데이터 가져오기 (Import)
| 항목 | 내용 |
|------|------|
| 가져오기 방식 | 파일 선택 또는 `.pongdang` 파일 열기 |
| 충돌 처리 | 기존 데이터 유지 + 신규 데이터 추가 (중복 스킵) |
| 복원 옵션 | 전체 덮어쓰기 / 병합 중 선택 |

### 백업 데이터 구조

```json
{
  "version": "1.0",
  "exportedAt": "2026-07-08T14:30:00Z",
  "appVersion": "1.0.0",
  "data": {
    "folders": [...],
    "items": [...],
    "settings": {...}
  },
  "images": ["images/xxx.png", ...]
}
```

### 기술 스택

| 용도 | 패키지 |
|------|--------|
| 로컬 DB | `sqflite` |
| 파일 경로 | `path_provider` |
| 고유 ID | `uuid` |
| 이미지 처리 | `image` |
| ZIP 압축 | `archive` |
| 파일 공유 | `share_plus` |
| 파일 선택 | `file_picker` |

### 저장 구조
```
/app_documents/
  /images/
    /originals/    # 크롭된 원본 이미지
    /thumbnails/   # 썸네일 이미지
  /database/
    pongdang.db    # SQLite DB 파일
  /backups/
    backup_001.json  # 자동 백업 (최근 3개 순환)
    backup_002.json
    backup_003.json
```

## Feature 6: 테마 설정 (Theme Settings)

| 항목 | 내용 |
|------|------|
| **기능 ID** | F6-THEME |
| **목적** | 사용자가 앱의 컬러 테마를 취향에 맞게 변경 |

### 제공 테마

| 테마 ID | 테마 이름 | 메인 컬러 | 느낌 |
|---------|----------|-----------|------|
| `peach` | 복숭아 우유 | `#FFAB91` | 포근하고 달콤한 (기본값) |
| `mint` | 민트 소다 | `#88D4AB` | 청량하고 시원한 |
| `lavender` | 포도 캔디 | `#B8A9C9` | 몽환적이고 달콤한 |

### 세부 기능

| ID | 기능명 | 설명 |
|----|--------|------|
| F6.1 | 테마 미리보기 | 설정 화면에서 테마별 미리보기 제공 |
| F6.2 | 테마 변경 | 테마 선택 시 즉시 앱 전체에 적용 |
| F6.3 | 테마 저장 | 선택한 테마를 로컬에 저장, 앱 재시작 시 유지 |

### 기술 요구사항

```dart
// 테마 Notifier (Riverpod)
class ThemeNotifier extends Notifier<String> {
  @override
  String build() => 'peach';

  void setTheme(String themeId) {
    state = themeId;
    _saveToPrefs(themeId);
  }
}

final themeProvider = NotifierProvider<ThemeNotifier, String>(ThemeNotifier.new);
```

| 용도 | 패키지 |
|------|--------|
| 상태 관리 | `flutter_riverpod` |
| 로컬 저장 | `shared_preferences` |

## 화면 구성 (Screen Flow)

```
[홈/메인 화면]
  ├── 전체 도감 그리드 뷰
  ├── 폴더 목록 탭
  └── 퐁당 버튼(+) → 펀치 모드 진입

[펀치 모드]
  ├── 이미지 선택 (카메라/갤러리)
  ├── 프레임 선택 바
  ├── 프레임 위치/크기 조절
  └── 크롭 버튼 → 등록 폼으로 이동

[등록 폼]
  ├── 크롭 이미지 미리보기
  ├── 라벨 입력 (태그 칩)
  ├── 별점 선택 (5점)
  ├── 메모 입력
  ├── 폴더 선택 (선택사항)
  └── 저장 버튼

[도감 상세]
  ├── 풀스크린 이미지
  ├── 메타데이터 카드
  └── 편집/삭제 버튼

[폴더 관리]
  ├── 폴더 목록 (트리 구조)
  ├── 폴더 생성/편집 모달
  └── 아이템 이동 인터페이스

[설정]
  ├── 테마 선택 (복숭아 우유/민트 소다/포도 캔디)
  ├── 데이터 내보내기/가져오기
  ├── 그리드 열 개수 설정
  └── 앱 정보
```
